We take security seriously

Luclky, because I made the backend I can tell you all about how it works!

This website make use of all the latest computer secutiry features that we learn about in the OHS Computer Science Club. Most importantly, iterative hashing. Rather than storing your password in so called "plain text" we store hashed versions of your password. This means that even if a hacker hacked our database, strong passwords would take years to break. Furthermore, our database is complex enough to not fall victim to SQL injection or cross site scripting attacks.
